# Curriculum

12 lessons. 4 tracks. Zero filler.

![Curriculum overview diagram](/images/curriculum.svg)

## Track 1 — Foundations

What an agent actually is. Not the hype version — the mechanical one.

1. **What is an agent?** — The difference between a chatbot and an agent. Tools, memory, goals.
2. **Tools and memory** — How agents call functions and what they remember between steps.
3. **Models** — Which model for which job. Cost, speed, and capability trade-offs.
4. **The tool surface** — Designing the set of actions an agent can take.

## Track 2 — Building

How to wire an agent together from scratch.

5. **Prompt architecture** — System prompts, context windows, and when each matters.
6. **Loops and stopping conditions** — How agents decide when they're done.
7. **Error handling** — What to do when the model hallucinates a tool call.
8. **Evaluations** — How to know if your agent is actually working.

## Track 3 — Shipping

Getting from "works on my machine" to running in production.

9. **Deployment patterns** — Async jobs vs. real-time agents. When to use each.
10. **Observability** — Logging, tracing, and debugging agent runs.

## Track 4 — Business

Using agents to run or grow a real operation.

11. **Agent economics** — When the cost of an agent run beats the cost of a human hour.
12. **Your first production agent** — End-to-end walkthrough of a real business automation.
