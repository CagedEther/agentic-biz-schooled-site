# Quick Start

The fastest way to learn how agents think is to watch one take apart something real.

![Quick start illustration](/images/quick-start.svg)

## How to use this

Describe your business idea or current operation in one paragraph. Be specific — the more detail you give, the more useful the output.

The agent will return:

1. **The customer** — who actually has this problem and how badly they have it
2. **The wedge** — the smallest version of this that someone would pay for today
3. **Unit economics** — the numbers worth questioning before you build anything
4. **What will probably kill it** — three realistic failure modes ranked by likelihood
5. **What to test this week** — one concrete experiment you can run in under a week

## What this is not

This is not a validation service. It's a thinking tool. The agent will be blunt. It will find the holes in your idea. That's the point.

If your idea survives the analysis, you'll have a sharper version of it. If it doesn't, you saved yourself months.

## Example prompt

> I want to build a SaaS tool that helps independent restaurant owners manage their social media. The idea is they describe a dish, and the agent writes and schedules posts across Instagram, TikTok, and Google. They pay $99/month.

Try something like that — and be ready for honest feedback.
